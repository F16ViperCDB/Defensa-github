## 🚀 Publicar en GitHub Pages
1. Sube este repo a GitHub (main branch).
2. Ve a **Settings → Pages**.
3. Selecciona la rama `main` y la carpeta `/ (root)` o `/docs` según prefieras.
4. Guarda. Tu sitio quedará en `https://tu-usuario.github.io/tu-repositorio/`.

> Nota: si quieres una sola página publicada, mueve el contenido de `demo-web` o `infografia` a la raíz del repo, o usa GitHub Pages por carpeta con un `index.html` en la raíz que enlace a ambas.
